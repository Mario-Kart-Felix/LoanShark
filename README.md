# Loan Shark
## Setup
* clone!
* `npm install -g brunch`
* `npm install`
* `bower install`
* `brunch watch --server`

## Brunch with Chaplin
![bwc-logo](http://brunch.io/images/svg/brunch.svg)

This is HTML5 application, built with
[Brunch](http://brunch.io) and [Chaplin](http://chaplinjs.org).
